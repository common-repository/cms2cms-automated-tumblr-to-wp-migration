Plugin for automated data migration from Tumblr to WordPress. Convert Tumblr to WordPress easily and fast.

== Description ==

Make a switch from your Tumblr website to WordPress in a few minutes with automated CMS2CMS converter. Migrate all your website content directly to WordPress with no codes at all.

= How does this plugin help: =
- Totally FREE Demo migration – to see how it works before the complete data migration;
- The price for migration is automatically estimated and depends on the amount of data you want to move from Tumblr to WordPress;
- Data mapping option, that allows you to choose what content to move to the new CMS and where to locate it;
- Complete Migration price depends on how many content items you want to move to your New WordPress website.


= WHAT WILL BE MOVED FROM TUMBLR TO WORDPRESS? =
* Pages along with the content included there;
* Attachments (images and videos);
* Images included in the content body.


= PRESERVED SEO JUICE AFTER THE MIGRATION =
* Migration of SEO friendly URLs;
* Migration of keywords, metadata/meta description;
* Automated 301 redirects from existing Tumblr site to the new WordPress website.


= COMPATIBLE WITH: WORDPRESS 4.1.X – 5.0.X. =
- Our support team is available live 24/7.
- For more customized migration type, CMS2CMS offers data migration service packages to order.

== Installation ==
1. Download the plugin zip file.
2. Go to the Admin Panel -> Plugins -> Add New -> Upload Plugin.
3. Choose the downloaded file in zip format and click the “Install Now” button.
4. Go to Admin -> Plugins, activate the “CMS2CMS: Automated Tumblr To WordPress Migration Plugin”.
5. Register an account or sign in to the existing one.
6. Provide your Existing website URL address.
7. Click ‘Verify Connection’. You’ll be redirected to the CMS2CMS website to proceed with your migration.
